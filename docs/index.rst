======
The Second State DevChain
======

This is the official documentation for the Second State DevChain. The DevChain is a Proof-of-Authority (PoA) blockchain designed for developers to get started with Lity language and virtual machine applications. The DevChain is free to all developers (i.e., it requires no cryptocurrency or gas).
